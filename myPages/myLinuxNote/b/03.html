    vi和vim编辑器

    大段 1/23/2022 linux

    # 基本介绍

    1. 所有的Linux系统都会内键vi文本编辑器
    2. vim具有程序编辑的能力，可以看作是vi的增强版本，可以主动地以颜色辨别语法的正确性，方便程序设计，代码补充，编译以及错误跳转等方便变成的功能特别丰富，在程序员中被广泛使用

    # vi和vim常用的三种模式

    1. 正常模式。以vim打开一个档案就直接进入一般模式了(这时默认的模式)，在这个模式中，你可以使用[ 上，下，左，右 ]按键来移动光标，你可以使用[ 删除字符 ]或[ 删除整行 ]来处理档案内容，也可以使用[复制，粘上]来处理你的我呢见数据
    2. 插入模式。按下[ i，l，o，O，a，A，r，R ]等任何一个字母之后才会进入编辑模式，一般来说，按[ i ]即可
    3. 命令行模式。在这个模式当中，可以提供你相关指令，完成读取，存盘，替换， 离开vim，显示行号等的动作则是在此模式中达成的

    # 快速入门

    ——使用vim开发一个Hello.java程序

    1. 【vim Hello.java】，回车

    2. 这时候进入编辑，想输入字符，输不进去，需要进入插入模式，输入【i】进入插入模式

    3. public Class Hello {
    public static void main(String[] args) {
        System.out.println("hello world!");
    }
}

    4. 输入【:】进入命令行模式，输入【wq】退出插入模式

    5. 输入【vim Hello.java】可以看到刚写的内容

    6. 输入【:q】退出不保存

    # vi和vim模式的相互切换

    1. 【vim xxx】 命令行 → 一般模式
    2. 【i】 一般模式→编辑模式
    3. 【ESC】 编辑模式 → 一般模式 命令模式→一般模式
    4. 【:】 一般模式 → 编辑模式
    5. 【:q】 退出，打开没做修改，用这个
    6. 【:q!】 强行退出，打开修改了不想保存，退出不保存
    7. 【:wq】 退出插入模式

    # 快捷键

    1. 拷贝当前行【yy】，拷贝当前行向下的5行并粘贴【5yy】,【p】粘贴
    2. 删除当前行【dd】，删除当前行向下的5行【5dd】
    3. 在文件中查找某个单词【:/关键词】回车查找，输入【n】就是查找下一个
    4. 设置文件的行号【:set nu】，取消文件的行号【:set nonu】
    5. 编辑/etc/profile文件，使用快捷键到底文档的最末行【G】和最首行【gg】，正常模式
    6. 在一个文件中输入“hello”，然后又撤回这个动作【u】
    7. 编辑/etc/profile文件，并将光标移动到20行【shift+g】。显示光标，输入20，输入shift+g
    8. ...
